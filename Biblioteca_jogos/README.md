<h3>DESCRIÇÃO DO PROJETO</h3>
Este projeto foi desenvolvido com o objetivo de praticar a criação de projetos utilizando tkinter, contendo opções de remover, inserir e alterar as descrições dos itens.

<h3>COMO EXECUTAR</h3>
<P>1. Baixe os arquivos do projeto.</P>
<P>2. Abra o arquivo e entre no arquivo "main"</P>
<P>3. Rode o arquivo.</P>
